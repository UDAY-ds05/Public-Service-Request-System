# 📌 Online Public Service Request System (PSRS)

The **Online Public Service Request System (PSRS)** is a web-based application designed to streamline the process of submitting, tracking, and resolving public service requests.

It bridges the gap between **citizens and government authorities** by providing a centralized, transparent, and efficient platform.

---

## 🚀 Key Features

### 👤 User Authentication & Role Management

* Secure login/signup system
* Role-based access (Citizen / Officer / Admin)

### 📝 Service Request Submission

* Citizens can raise complaints or service requests
* Categorization of issues (Water, Electricity, Roads, etc.)

### 📊 Dashboard (SPA-based UI)

* Dynamic navigation without page reload
* Sections like Dashboard, Profile, Track Requests

### 🔍 Request Tracking System

* Real-time status updates (Pending, In Progress, Resolved)
* Easy tracking of submitted complaints

### 🧑‍💼 Officer Panel

* View and manage assigned requests
* Update status and provide resolutions

### 🔔 Notifications & Updates

* Inform users about request progress

---

## 🧠 Core Concepts Used

* **Full Stack Web Development**

  * Frontend: HTML, CSS, JavaScript
  * Backend: Java Servlets & JSP
  * Database: MySQL (JDBC)

* **MVC Architecture**

  * Separation of UI, business logic, and data

* **Session Management**

  * Secure login using HTTP sessions & cookies

* **Database Design**

  * Entities: User, Request, Officer, Status Tracking

* **CRUD Operations**

  * Create, Read, Update, Delete functionality

* **Asynchronous UI**

  * Dynamic content loading for smooth UX

---

## 🏗️ System Workflow

1. User registers/logs in
2. Submits a service request
3. Request is assigned to an officer
4. Officer processes and updates status
5. User tracks progress in real-time
